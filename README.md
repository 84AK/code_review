# AI Code Review Pro (2026 Edition)

학생들이 제출한 MBTI 및 POSE 게임 코드를 AI가 자동으로 분석하고 채점 및 피드백을 제공하는 프리미엄 웹앱입니다.

## 🚀 주요 기능
- **MBTI 코드 분석**: HTML/CSS/JS 기초 프로젝트의 완성도와 코드 품질 평가.
- **POSE 게임 분석**: Teachable Machine 포즈 인식 로직 및 게임 밸런스 평가.
- **Bento Grid Dashboard**: 2026년 최신 트렌드를 반영한 세련된 관리자 대시보드.
- **Gemini 3.1 AI**: 최신 생성형 AI 모델을 통한 정교한 코드 리뷰 및 등급 산정.
- **Bulk Upload**: 여러 개의 .zip 파일을 한 번에 업로드하여 병렬 처리 가능.

## 🛠 기술 스택
- **Base**: Vanilla HTML5, Modern CSS (Native Nesting), Vanilla JavaScript (ESM)
- **AI**: Gemini 3.1 Flash / Pro
- **Libraries**: JSZip (File Extraction), Lucide Icons (Visuals)
- **Design**: Glassmorphism 2.0, Bento Grid Layout

## 📖 사용 방법
1. 브라우저에서 `index.html`을 엽니다.
2. 우측 상단 **설정** 버튼을 클릭하여 **Gemini API Key**를 입력합니다.
3. 분석할 카테고리(MBTI 또는 POSE)를 선택합니다.
4. 학생들의 `.zip` 파일을 대시보드 중앙에 드래그 앤 드롭합니다.
5. 분석 결과를 확인하고 등급(ABCD) 및 피드백을 검토합니다.
